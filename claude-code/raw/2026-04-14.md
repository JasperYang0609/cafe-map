## 2. V19 測試報告處理

### Jasper 回報的 6 個 bug：

**Bug 1**: 免費用戶用完 3 次後，點種子 → 等廣告載入 → 看廣告，跳出「廣告準備中」，沒有店家出現，連續 5 次都沒有抽卡成功
- 根因：`initAds()` 失敗後 `initStarted=true` 永不重置；`EARNED_REWARD` 事件在某些 SDK 版本不觸發
- 修復：`initStarted = false` on failure；新增 `AdEventType.OPENED` 作為 fallback，`finish(earned || shown)`
- 檔案：`src/lib/ads.ts`

**Bug 2**: 探索頁、搜尋紀錄頁、收藏頁底下廣告 banner 沒出現
- 根因：`BannerAdPlaceholder` 沒呼叫 `initAds()`；`useMemo` 快取 JSX 失敗後無法重試
- 修復：移除 `useMemo`，新增 `useEffect` 呼叫 `initAds()`，新增失敗重試機制（MAX_RETRIES=3, RETRY_DELAY=5000）
- 檔案：`src/components/BannerAdPlaceholder.tsx`

**Bug 3**: 未訂閱用戶按收藏 → 看廣告 → 收藏失敗
- 根因：同 Bug 1
- 修復：同 Bug 1

**Bug 4**: 地圖 pin 點擊後不變紅色
- 根因：Android 上 `pinColor` 動態變更不觸發原生 marker 重繪
- 修復：非收藏 marker key 加入 selected 狀態強制 remount
- 檔案：`app/(tabs)/map.tsx`

**Bug 5**: 收藏 emoji icon 點擊時鄰近 icon 閃爍
- 根因：所有收藏 marker `tracksViewChanges={true}` 導致全部重繪
- 修復：`tracksViewChanges` 僅 selected marker 開啟
- 檔案：`app/(tabs)/map.tsx`, `app/(tabs)/favorites.tsx`

**Bug 6**: 訂閱用戶登出後 banner 不顯示
- 根因：BannerAdPlaceholder 沒有監聽 `isSubscribed` 變化初始化 AdMob
- 修復：`useEffect` 監聽 `isSubscribed` 變化並呼叫 `initAds()`
- 檔案：`src/components/BannerAdPlaceholder.tsx`
